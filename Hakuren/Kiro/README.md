# Kiro Setup

## Install
**Windows (PowerShell):**
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

## Login
```cmd
kiro-cli login
```

### Kiro Free tier
$0 per month **50 credits** + **Welcome bonus 500 credits** (expires in 30 days)

## Tip
- Kiro Permission Always Accept: `kiro-cli --trust-all-tools` ( in chat. `/tools trust-all` , `/acceptall`)
- Use cheap models: `auto:1.0x credits`, `glm-5:0.5x credits`
- Plan Mode: `/plan`, `shift + tab`
